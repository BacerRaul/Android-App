Aplicatie realizata in Android Studio pentru gestiunea de documente legate de scoala.

Elevii / Studentii isi pot adauga materii. La fiecare materie isi pot adauga notitele si notele ce tin de aceasta.
Notitele pot fi adaugate fie prin intermediul camerei, scanand / pozand foi, sau prin incarcarea acestora din telefon.
De asemena, pot cere ajutorul unui AI integrat: explicarea notitelor sau crearea de teste grila cu raspunsuri pe care pot exersa.
